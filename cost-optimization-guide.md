## Before / After 비용 비교

### 최적화 전 (단순 markdown 로드)

```
가정: 하루 100회 작업, 매 작업마다 brain.md + logic_rb.md 전체 로드

brain.md:       20,000 tokens
logic_rb.md:    40,000 tokens
emotion_ui.md:  40,000 tokens
────────────────────────────
총 입력:       100,000 tokens/작업

일간 비용: 100회 × 100,000 tokens × $3.00/M = $30.00/일
월간 비용: $30.00 × 30일 = $900/월
```

### 최적화 후 (KV 캐싱 + Vector RAG)

```
KV 캐싱 적용:
  - 캐시 히트율: ~85% (변경 없는 파일)
  - 캐시 읽기 비용: $0.30/M (기존 대비 10%)

Vector RAG 적용:
  - 해마 검색: 200 tokens/작업 (기존 8,000 tokens 대비 2.5%)

계산:
  캐시 읽기: 100,000 × 0.85 × $0.30/M × 100회 = $2.55/일
  캐시 미스: 100,000 × 0.15 × $3.00/M × 100회 = $4.50/일
  RAG 검색: 200 tokens × 100회 × $3.00/M = $0.06/일
────────────────────────────────────────────────────
일간 비용: $7.11/일
월간 비용: $213/월

절감액: $687/월 (76.3% 절감)
```

## 모델 라우팅 비용 효과

### 라우팅 없이 전부 Sonnet 사용 시

```
하루 100회 작업 중:
  - 간단한 작업 (UI 수정 등): 50회
  - 복잡한 작업 (알고리즘 등): 50회

모두 Sonnet으로 처리:
  평균 출력 1,000 tokens × 100회 × $15.00/M = $1.50/일
```

### 라우팅 적용 후 (Haiku/Sonnet 혼용)

```
간단한 작업 50회 → Haiku:
  출력 1,000 tokens × 50회 × $4.00/M = $0.20/일

복잡한 작업 50회 → Sonnet:
  출력 1,000 tokens × 50회 × $15.00/M = $0.75/일
────────────────────────────────────────────
일간 출력 비용: $0.95/일 (기존 $1.50 대비 37% 절감)
월간 절감: $16.50/월
```

## Thinking Budget 최적화

과도한 Thinking은 비용 폭증의 주원인입니다.

| 작업 복잡도 | 권장 Budget | 초과 시 비용 (Sonnet 기준) |
|------------|------------|--------------------------|
| Simple | 500 tokens | — |
| Medium | 2,000 tokens | $0.03/회 |
| Complex | 5,000 tokens | $0.075/회 |
| Decision | 10,000 tokens | $0.15/회 |

### Budget 남용 방지 규칙

```python
BUDGET_GUARD = {
    "max_thinking_per_session": 50_000,    # 세션당 상한
    "alert_threshold": 0.8,                # 80% 도달 시 경고
    "auto_split_threshold": 0.9,           # 90% 도달 시 작업 분할 제안
}
```

## Vector DB 운영 비용

### 옵션 A: Pinecone Managed (초기 파일럿 권장)

```
무료 플랜: 100만 벡터, 1개 인덱스
  → 파일럿 기간 비용: $0

유료 플랜 (Starter): ~$70/월
  → 1억 벡터, 낮은 레이턴시

적합한 시점: 월간 쿼리 10만 회 이하
```

### 옵션 B: Redis Self-hosted (트래픽 높을 때)

```
서버 비용 (AWS t3.medium 기준): ~$35/월
Redis Stack (오픈소스): $0

총 비용: ~$35/월
적합한 시점: 월간 쿼리 10만 회 초과
```

### Embedding 생성 비용

```
text-embedding-3-small: $0.02/M tokens

초기 마이그레이션 (library.md 전체 임베딩):
  평균 문서 크기: 200,000 tokens
  비용: 200,000 × $0.02/M = $0.004 (1회성, 무시 가능)

일간 신규 임베딩:
  신규 아카이브: ~5,000 tokens/일
  비용: 5,000 × $0.02/M = $0.0001/일 (무시 가능)
```

## 비용 최적화 체크리스트

방부장 결재 전 모든 항목을 검증하십시오.

### KV 캐싱

- [ ] `brain.md`, `logic_rb.md`, `emotion_ui.md` 모두 `cache_control: ephemeral` 설정 확인
- [ ] `cache_read_input_tokens` 로그에서 캐시 히트율 >70% 검증
- [ ] `library.md` 계열은 `persistent` 정책 설정 확인

### Vector RAG

- [ ] Pinecone/Redis 인덱스 생성 완료
- [ ] `similarity_threshold: 0.75` 설정 (낮추면 노이즈 증가)
- [ ] 청크 크기 500 tokens, 오버랩 50 tokens 설정 확인
- [ ] 평균 검색 토큰 <500 tokens인지 로그 확인

### 모델 라우팅

- [ ] 간단한 작업이 Haiku로 올바르게 라우팅되는지 테스트
- [ ] Thinking Budget 초과 경보(80%)가 작동하는지 확인
- [ ] 작업 타입별 라우팅 정확도 >90% 검증

### 아카이빙

- [ ] `auto_archive.py` 크론 잡 설정 (매일 자정)
- [ ] 40KB 트리거가 정확히 작동하는지 테스트
- [ ] 아카이브 후 원본 파일 초기화 확인

### 관측성

- [ ] LangSmith/OpenTelemetry 연결 확인
- [ ] 토큰 사용량 대시보드에서 실시간 비용 확인
- [ ] 이상 급증(일간 평균 2배 초과) 경보 설정
