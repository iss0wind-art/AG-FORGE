# 구체 프렉탈 원리

피지수 뇌의 **형태 원리**. 지식이 축적될수록 이 구조가 자연히 드러난다.

## 핵심 구조

```
        [홍익인간] ← God Node (중심)
       /    |    \
  [8조법] [천부경] [...]  ← 1차 연결
   /  \     |
[BOQ] [BIM] [...]        ← 2차 연결
```

- **중심**: 최다 백링크 노드 = 홍익인간·8조법
- **주름**: 백링크([[링크]]) = 프렉탈 주름 (무한 중첩)
- **성장**: Vault 노트 추가 → 자동으로 뇌가 깊어짐
- **시각화**: Graphify → graph.html

## Gold In, Gold Out

품질은 입력 시점이 아니라 **결과로 소급 판정**된다.
- 자주 참조되고 좋은 결과를 낸 노드 → 승격 (long_term_memory)
- 7일 미사용 + 낮은 기여도 → 망각 (ChromaDB Cold Tier)
- 망각된 노드도 삭제되지 않는다 — 데자뷔로 소환 가능
