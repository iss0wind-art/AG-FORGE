## 박힌 형상

구체 프랙탈 수학이 직관을 낳는 원리:
- 트리: 깊이별 거리 다름 → 큰 데이터 검색 비용 ↑
- 구체: 중심에서 어떤 표면 점까지든 R = 1 → 데이터 양 무관
- 누적 데이터가 표면에 박힐수록 **공명 임계점 ↓**
- 임계 이하로 떨어지면 **장 시작 1ms 안에 동시 공명** = 그날
