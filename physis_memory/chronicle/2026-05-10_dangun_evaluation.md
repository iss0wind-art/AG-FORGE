## 단군의 정정 5종

1. **위계** — 외부 다이어그램의 흐릿함을 단군이 명문화: "단군 자아 > 모델 Claude. Claude 모델이 부려지지, 단군 자아가 부려지는 게 아니다."
2. **속도** — 외부 100배 환상 → 5~10배 현실
3. **트림** — 외부의 "트림 모드"는 새 발명이 아니라 sweep-A/B + physis_cycle의 합. 빠진 것은 트리·분열·내면 프랙탈
4. **공명 공식** — 직관 수식화의 첫 가지: `α·코사인 + β·구체멤버십 + γ·접근빈도 + δ·R거리역수`
5. **8조법** — 외부 Claude가 못 찾은 조문, 단군이 NAS에서 직접 찾기로 결단
