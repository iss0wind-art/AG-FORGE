## 3. ★ 결정적 통찰 4가지

### 통찰 1 — 한온시스템 BUY의 결정적 정합 (★★★★★)

```
4지국: 7번 일관 BUY 72-76% / TV: STRONG_BUY / 당일: +21.80%
```

**5/11 가장 큰 정합 신호**. 7회 반복 분석 모두 BUY로 일치 + TV·시장 흐름 3중 정합. 만약 실거래였다면 일중 +20% 수익. KIS_MOCK이라 모의만이지만 **패턴 학습 가치 ★★★★★**.

이게 [[바람을_보다_비정형의_시선]] 2단계 "자기만의 결" 형성의 첫 정합. 단군 공명 공식 α·β·γ·δ가 살아 움직이기 시작하는 자리.

**[[2026-05-11_failure_as_seed|자각 10 "실패가 성공의 씨앗"]]의 본 검증** — 4지국 SELL 신호 4건 중:
- 3건 완전 충돌 (시장은 상승)
- 1건 부분 정합 (방향 일치, 폭은 약함)
- **SELL 정합도 약 12.5%** (4건 중 0.5건)

이건 [[5_9_거래결정_30건_정독_outcome_score_시드]] 분석에서 짚었던 "SELL 신호 약점"의 본격 검증. 패턴이 일관됨 — **상승 추세에서 RSI 70+ 과매수 신호로 SELL 판단 → 시장은 더 상승**.

### 통찰 3 — RSI 과매수 패러독스

```
삼성전자  RSI 78.8 → +7.26%
TIGER반도체 RSI 86.0 → +6.42%
한온시스템 RSI 71.8 → +21.80%
진원생명  RSI 60.1 → +5.58%
대한전선  RSI 77.2 → -3.32%
KODEX2차전지 RSI 63.1 → -3.98%
```

**RSI 70+ "과매수" 신호가 SELL의 정확한 트리거 아님**. 강세장에서는 RSI 80도 계속 상승. 4지국의 SELL 판단이 RSI를 **과대 가중**할 가능성.

다음 진화 후보:
- 단군 공명 공식 α·β·γ·δ에서 RSI 가중 재학습
- 추세 강도(ADX) + 거래량 가중 보강
- 또는 RSI 단독이 아닌 **이평선 + 거래량 + RSI 3중 일치만 SELL 트리거**

### 통찰 4 — HITL 차단의 가치 입증 (안전장치 #4)

```
12:10 사이클     → 방부장 직접 결단 → 4건 approved
                  (SELL 2건·BUY 2건 — KIS_MOCK이라 실거래 0)
13:55~15:32 사이클 → HITL timeout 자동 보류 (방부장 일하느라)
                  → SELL 신호 4건 (대한·삼성×2·14:56 삼성) **자동 미실행**
```

방부장이 일하느라 응답 못한 자리에서 **자동 보류가 결과적으로 손실 방지** — SELL 신호가 틀렸으니 미실행이 정답.

[[7중_안전장치]] §4 HITL 5분 무응답 → 자동 보류의 정확한 작동. 인간 직관·결단의 불완전을 자동 보류가 백업.

## 4. 자기 분석·직관 형성의 시드

[[바람을_보다_비정형의_시선]] 2단계 진입의 명시적 시드 정형화:

### A. 한온시스템 BUY 패턴 = 알파 (가중 ↑)
- 7회 반복 분석 모두 BUY 72-76% 일관
- 합의율 100% 거의 모든 사이클
- TV STRONG_BUY (16·0·10) 압도적 매수 신호
- 당일 +21.80% 검증

→ **다음 사이클에서 한온시스템 BUY 신호 강하게 신뢰** + 유사 패턴(2단계 power_infra·datacenter) 종목 자동 가중

### B. SELL 4건 패턴 = 약점 (가중 ↓)
- RSI 70+ 과매수에서 SELL → 강세장에서 모멘텀 무시
- 합의율 100% 일관이어도 시장과 충돌
- HITL 차단된 것이 다행

→ **다음 사이클에서 RSI 단독 SELL 신호 회의적으로 받아들임** + 추세 강도(ADX)·거래량 3중 일치 필요

### C. 종목 풀의 좁음 = 다음 진화 영역
- 5/11 분석된 종목: 6종 (반복 분석 포함)
- 그 중 12:10 사이클의 4종(TIGER·KODEX·한온·진원) = 다양화 시도
- 13:55~ 사이클은 3종 반복

→ **NewsScraper·StockScreener 후보 풀 다양화** 또는 4지국 김육 강역에서 결단

## 5. 단군 공명 공식 α·β·γ·δ 재학습 시드

[[단군의_평_2026-05-10]] 박은 공식:
```
공명점수(node, query) = α·코사인유사도 + β·구체멤버십 + γ·접근빈도 + δ·R거리역수
```

5/11 분석에서 발견된 시드:
- **한온시스템 BUY 패턴** → β(구체멤버십) ↑ (PHASE_2 power_infra의 일관 정합)
- **SELL 4건 충돌** → α(코사인유사도) 재계산 — RSI 단독 코사인 약화
- **HITL 차단 사이클** → γ(접근빈도) ↑ (자동 보류 빈도가 가중치 시드)
- **방부장 12:10 결단** → δ(R거리역수) ↑ (직접 결단 사이클의 가중치 시드)
