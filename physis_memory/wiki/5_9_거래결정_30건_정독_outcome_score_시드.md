# 5/9 거래 결정 30건 정독 — outcome_score 시드

[[바람을_보다_비정형의_시선]] 2단계 "자기 분석·직관 형성"의 첫 가지. 5/9 4지국 파이프라인이 박은 30건 거래 결정의 통계적 시그니처 정독. 이것이 outcome_score 사이클의 출발점이며, 단군 공명 공식 α·β·γ·δ가 살아 움직이기 시작할 토양.

## 1. 통계 시그니처

### 종목 분포 (반복 분석)
```
대한전선   (001440) — 10건 / 30 (33%)
한온시스템 (018880) — 10건 / 30 (33%)
삼성전자   (005930) — 10건 / 30 (33%)
```

**3 종목만 반복 분석**. 다양성 낮음. 5/9가 4지국 파이프라인의 첫 본격 운영일이라 종목 풀이 작았던 흔적. 다음 사이클에서는 [[섹터_4단계_가중치_코드]]의 매핑된 16 종목으로 다변화 필요.

[[섹터_4단계_가중치_코드]] 기준 분류:
- 대한전선 = **2단계 power_infra** (가중치 0.95)
- 한온시스템 = **2단계 datacenter** (가중치 0.95)
- 삼성전자 = **1단계 semi** (가중치 0.85, 선반영 종료)

→ **1·2단계 종목만 분석됨**. 4단계 바이오·로봇 종목 0건. 정보 비대칭의 결을 잡는 자리는 아직 진입 안 함.

### 시그널 분포
```
HOLD — 18건 (60%)
BUY  —  6건 (20%)
SELL —  6건 (20%)
```

**HOLD 60% = 합의율 60% 미달**. 7중 안전장치의 자연스러운 결과.

### 게이트 통과
```
gate_passed=False — 18건 (60%)
gate_passed=True  — 12건 (40%)
```

**게이트 차단 60%**. 신뢰도 70% × 합의율 60% 게이트가 약 40%만 통과시킴. **이게 [[7중_안전장치]]의 본 의도** — "안 하는 결정"이 곧 보호.

### 시장 감성
```
긍정 — 28건 (93%)
중립 —  2건  (7%)
```

5/9는 시장 감성이 거의 긍정 일관. EnsembleAgent가 시장 환경을 긍정으로 판정한 날.

## 2. 정합 점검 — [[7중_안전장치]] 작동 검증

| 안전장치 | 5/9 baseline |
|---------|--------------|
| 신뢰도 70% 게이트 | ✅ 작동 (60% 차단) |
| 합의율 60% 게이트 | ✅ 작동 (60% 차단 일부) |
| HOLD 자동 거절 | ✅ 18건 HOLD = 자동 보류 |
| 일일 3건 한도 | ✅ 종목 수 자체가 3 (한도 내) |
| 100만원 한도 | (체결 0건 → 검증 못 함) |
| KIS_MOCK 기본 | ✅ 실손실 0 |
| HITL 5분 자동 보류 | (체결 0건 → 검증 못 함) |

**0건 체결 = 안전장치 정상 작동의 증거**. 워런 버핏 룰 1 "절대 잃지 마라"의 자동화.

## 3. outcome_score 누적 한계

현 30건은 **결과 평가 불가**:
- 실제 거래 0건 → 실손익 0
- outcome_score = 0 (초기값)
- 단군 공명 공식 α·β·γ·δ 가중치는 아직 1.0:1.0:1.0:1.0 시드 상태

**진짜 가중치 학습이 시작되는 시점**:
- 1차 10% 목표 도달까지의 6~12개월 동안 누적
- 이긴 거래 vs 진 거래 비율
- 합의 통과한 12건 중 실제로 시장이 동의한 비율 (BUY 6건 / SELL 6건 결정의 결과)

## 4. 직관 시드 (1단계 → 2단계 전환)

[[바람을_보다_비정형의_시선]] 1단계 학생에서 2단계 직관으로 가는 다리:

```
정형 학습 (지금)               2단계 직관
─────────────                  ─────────────
HOLD = 합의율 60% 미달  →  "같은 종목이 3번째 분석에서도
                            HOLD면 → 그 종목 자체의
                            결이 그날 닫혀있다는 신호"

긍정 시장 감성 28/30   →  "긍정 일관 일에 HOLD가 60%면
                            진짜 매수 기회가 없는 게 아니라
                            매수 압력이 이미 선반영됐다는
                            결"
```

이것이 직관의 시드. outcome_score 누적되면서 가중치 재학습으로 검증됨.

## 5. 비정형 시선이 들어올 자리 (3단계 진화)

[[바람을_보다_비정형의_시선]] 3단계 진입의 자리:

- **같은 3 종목 반복 분석** → 4단계 바이오·로봇으로 종목 풀 확장 시 정보 비대칭의 결을 잡는 자리
- **HOLD 18건의 reasoning 정독** → "왜 HOLD였는가"의 비정형 패턴
- **gate_passed=True 12건의 reasoning** → 실제 통과한 신호의 결 (12건이 모두 진짜 알파였는지)
