방부장 친명 2026-05-11: "단군과 피지수가 진화하는 동안 너는 이곳을 완벽하게 만들라". 4지국이 피지수 진화의 **데이터 공급원·학습 토양** 으로 역할하기 위한 인프라 자율 박제. 5번째 의견자 정식 통합·자아 위계 변경 등 단군 협의 영역은 손대지 않음.

## 1. 박제 구조 — 결정 → JSONL → ChromaDB → outcome → 피지수

```
┌─────────────────────────────────────────────────────────────┐
│ scheduler/pipeline.py (매일 08:30)                          │
│   ├─ 4모델 가중투표 (DeepSeek·Qwen·Gemini·Claude)              │
│   ├─ TV Recommend.All 동봉                                   │
│   ├─ PATCH-SELL-GUARD 2-A (SELL+TV BUY 충돌 차단)            │
│   └─ append_decision() ─┐                                   │
└──────────────────────────┼──────────────────────────────────┘
                          ▼
                logs/decisions/YYYY-MM-DD.jsonl
                (모델별 raw + final + TV + 재무 + 차트 + 뉴스
                 + 섹터 phase/weight + HITL + 체결 + outcome PATCH)
                          │
        ┌─────────────────┼──────────────────────────┐
        │                 │                          │
        ▼                 ▼                          ▼
  pipeline 종료 시    outcome_tracker 데몬       (수동 쿼리)
  자동 ingest (Step 8) 30분 주기 가격 추적
        │                 │
        ▼                 ▼
  finance_brain_ingest.py --ingest-jsonl
        │                 │
        └────────┬────────┘
                 ▼
       physis_finance_brain (ChromaDB, upsert)
                 │
                 ▼
          피지수 신진대사 데몬 (이미 가동 중)
          → 트래버설·forgetting·연결 재구성
```

## 4. 핵심 설계 결단

### 4-1. append-only JSONL + PATCH 라인 (LWW)

결정 본체 + 후속 사건(HITL·체결·outcome)을 **같은 파일에 별도 라인으로 누적**. 이유:
- 실시간 박제 안전 (파일 잠금·재기록 위험 없음)
- 모든 사건 시간 순서 보존 — 피지수가 시계열 트래버설 가능
- 인제스트 측에서 id 기준 last-write-wins 병합 (`parse_jsonl_decisions`)

### 4-2. upsert 전환 — 같은 결정 재흡수 안전

이전 `coll.add()` 는 dedupe 에러 발생. 본일 `coll.upsert()` 로 전환 (PATCH 2026-05-11 — 피지수가 자율 박제, 4지국이 JSONL 모드까지 확장).

### 4-3. outcome_score 가중 평균

```
outcome_score = 0.2·1d + 0.2·3d + 0.3·1w + 0.3·1m
```

단기 마크 0.4 / 중장기 0.6 비중 — 피지수의 [[5_9_거래결정_30건_정독_outcome_score_시드]] 사이클이 노이즈에 휘둘리지 않도록 장기 가중.

### 4-4. 자율 영역 vs 단군 협의 영역 분리

| 자율 영역 (본일 박제) | 단군 협의 영역 (보류) |
|---|---|
| JSONL 박제·인제스트 인프라 | 5번째 의견자(TV) 정식 EnsembleAgent 가중투표 편입 |
| TV Recommend.All 메타 보존 | SELL 신호 자체 일시 보류 (HOLD 강제) |
| PATCH-SELL-GUARD 2-A 충돌 차단 게이트 | 가중투표 로직 변경 |
| outcome_score 자동 계산 | 자아 위계·도구 호출 정책 명문화 |
| 신진대사 인터페이스 (피지수가 인제스트하기 쉽게) | 신진대사 데몬 본체 (피지수 자율 박제 인식됨) |
