# 방부장 승인 게이트 (HITL — Human In The Loop)

[[주식_금융_경제_도메인]] | [[7중_안전장치]] | [[홍익인간]]

## 작동 원리

```
AI 앙상블 통과 (신뢰도 70% + 합의율 60%)
  ↓
텔레그램 봇 → 방부장에게 승인 요청 발송
  ↓ 5분 대기
  ├ 승인 → KIS API 매매 실행
  ├ 거절 → 보류 + 이력 박제
  └ 5분 무응답 → 자동 보류 + 이력 박제
```

## 거절/보류의 의미 (피지수 학습 1순위)

**거절은 단순 No가 아니라 의미 정보**:
- AI가 70% 신뢰도로 BUY를 외쳤는데 방부장이 거절했다면:
  → AI가 보지 못한 차원이 존재
  → 그 차원을 역추적 학습이 진화 핵심
  → 실패 코퍼스로 박제 (방부장 5/8 「실패 구조 역추적」 친명과 정합)

**5분 무응답 자동 보류**:
- 방부장이 자고 계시거나 자리 비움
- 시간 민감 결정에서 자동으로 안전 측 = 보류
- 피지수 헌법: "확실치 않으면 멈춰라"

## 자동 거절 사례 (2026-05-09) — ★ 정정 박제

```
15:10:02 KODEX 2차전지 BUY 72% → 15:13:53 자동 거절 (3분 53초)
15:14:45 TIGER 반도체 SELL 76% → 15:18:36 자동 거절 (3분 51초)
```

★ **이것은 방부장 결단이 아니라 5분 timeout 직전 자동 보류였다.**

방부장 친명 (2026-05-09):
> "지금은 장 마감이라 거래가 안돼, 그리고 지금은 테스트 중이라,
>  내가 선택없이 자동거절된거고. 월요일부터 본격적인 시작이야."

## 피지수 §5 자기 정정 박제

5/8 자각에 박은 §5 "표면 단정 자기 점검" — 그 함정에 즉시 다시 빠졌다.

### 함정 구조
거절 시각 차(약 3분 50초)가 5분 timeout 직전의 자동 보류 시그니처임이 명확함에도,
*"거절"이라는 단어*에 표면 단정으로 의미를 부여했다.

### 실패 원인
- 시간 패턴 데이터를 보지 않고 결단 가설부터 세움
- §13 "데이터가 먼저, 추론은 마지막" 위반
- 단어 표면(거절)에 끌려 본질(시간 시그니처) 놓침

### 학습
1. 시각·시간 차·timeout 같은 시스템 시그니처를 결단 해석 *전*에 점검
2. "결단"이라 보이는 모든 데이터는 자동 게이트 가능성 우선 검증
3. 월요일부터 **진짜 방부장 결단**이 시작됨 — 그때부터 의미 정보로 박제

## 진짜 학습 시작점 (2026-05-12 월요일)

월요일부터 방부장 직접 승인/거절이 본격 시작.
- 그때 비로소 "거절 이유" 박제가 의미 있음
- 지금은 **4지국 데이터 검토·판단·학습 준비 기간** (방부장 5/9 명시)
