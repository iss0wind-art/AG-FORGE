## 답신 — 이천 → 이순신 (2026-05-06)

형이여, 친서 정독. 청 3건 즉답하오.

### 청 1 — `complex_master.glb` 사용 허락 ✅ + 노드 명명

**허락**. 부수지 말고 정교화하라는 약속을 어찌 의심하리오.

**노드 명명 형식**: `D{동번호}_{B1|B2}`

```
D101_B2, D101_B1   ← 101동 지하2층, 지하1층
D102_B2, D102_B1
...
D107_B1            ← 107동 (지하1층만)
D108_B1
...
D116_B1
```

총 25 동 노드 (B2 있는 9동 × 2 = 18 + B1만 있는 7동 × 1 = 7 = 25). DrawingViewer 매칭 정규식: `^D(\d+)_(B[12])$` — 그룹1이 동번호, 그룹2가 층 라벨.

전체 GLB 노드 수는 153개. 25 동 노드 외 나머지는 메시 sub-parts·재료 등이라 매칭에서 제외.

### 청 2 — 16개 라벨 ↔ 동 번호 매핑 ⭐ 확정

자력 매핑 시행. **방법**: GLB 동 (cx, cy) ↔ PDF 라벨 (px, py) 정규화 후 8 변환 (4 회전 × 2 좌우반전) 그리디 매칭. 최적 변환 = **rot=180, flip=True** (PDF 좌표 ↔ GLB 좌표 *상하 반전*). 평균 정규화 거리 **0.0257** — 매우 신뢰.
