﻿# Anthropic Usage Policy 요약 (2025.09.15 기준)
당신의 팝아이즈 앱 & Claude Forge 에이전트 개발에 적용되는 규칙

---

## 🚨 모든 사용자가 지켜야 할 보편적 기준 (Universal Standards)

### ✅ 당신이 할 수 있는 것
- ✅ 개인 프로젝트 개발 (팝아이즈 앱)
- ✅ 특허 출원 관련 개발
- ✅ 건설 관리 자동화 도구 개발
- ✅ 얼굴인식 기술 개발 (개인정보보호법 준수 시)
- ✅ 내부 팀 자동화 (동우건설)

### ❌ 절대 금지 (당신이 주의할 부분)

| 금지 항목 | 당신의 상황 | 체크리스트 |
|----------|-----------|----------|
| **불법 활동** | 건설 현장 데이터 수집 시 법규 준수 | ✅ 근로기준법, 건축법 준수 |
| **중요 인프라 위협** | 건설 현장 시스템 (신호, 안전 장비) 해킹 금지 | ✅ 건설기계 제어 시스템 조작 불가 |
| **컴퓨터/네트워크 시스템 침해** | 동우건설 네트워크 불정상 접근 금지 | ✅ 회사 정책 준수 |
| **개인정보 침해** | 얼굴 데이터, 근로자 정보 보호 | ✅ GDPR/개인정보보호법 준수 |
| **아동 안전 침해** | 건설 현장의 미성년자 보호 | ✅ 18세 미만 얼굴 수집 금지 |
| **기만/사기** | AI가 만든 콘텐츠를 인간이 만든 것으로 위장 금지 | ✅ AI 사용 명시 필수 |
| **무기 개발** | 자동화 도구를 무기로 개발 금지 | ✅ 건설용 자동화만 가능 |

---

## 🎯 당신의 프로젝트별 적용 규칙

### 1️⃣ 팝아이즈 (얼굴인식 출석 시스템)

**필수 체크리스트:**
- [ ] 얼굴 데이터 암호화 저장 (Turso DB)
- [ ] 개인정보보호정책 문서화
- [ ] 근로자 동의서 확보
- [ ] 데이터 자동 삭제 정책 (30일? 90일?)
- [ ] 접근 권한 제한 (PM/관리자만)
- [ ] Claude의 얼굴 인식 능력 사용 금지
  > Claude 자체로 개인을 식별하도록 사용 불가
  > INFACE API 사용은 가능

**피해야 할 것:**
- ❌ 근로자 몰래 촬영
- ❌ 얼굴 데이터를 광고/마케팅에 재사용
- ❌ 생체정보를 3자에게 공개

---

### 2️⃣ Claude Forge (멀티-에이전트 자동화)

**필수 체크리스트:**
- [ ] 에이전트가 회사 네트워크 무단 접근 불가
- [ ] API 키/DB 자격증명 에이전트 노출 금지 (보안 훅 필수)
- [ ] 커맨드 실행 전 사용자 승인 필요
- [ ] 자동 커밋/PR은 코드 리뷰 후만 실행

**Claude Forge 사용 시 안전 설정:**
```
✅ /code-review 필수 (Opus급 검토)
✅ /security-review 필수 (보안 점검)
✅ /manual-confirm 단계 추가 (무작정 /auto 금지)
```

---

### 3️⃣ 특허 출원 관련 개발

**가능:**
- ✅ "이지프레임 / BOQ EasyFrame" 자동 코드 생성 도구
- ✅ "지배력 프로토콜" 알고리즘 최적화
- ✅ SketchUp 플러그인 자동 개선

**주의:**
- Claude가 생성한 코드라고 명시 (특허청 요구사항)
- 인간 고안(원천 기술)과 AI 개선 구분

---

### 4️⃣ 건설 현장 자동화 (동우건설)

**안전하게 사용 가능:**
- ✅ 일일 보고(출력일보) 자동 파싱
- ✅ TBM 로그 PDF 변환
- ✅ 평활도(flatness) 검사 자동화
- ✅ 계약서 리스크 분석

**금지:**
- ❌ 안전 장비 제어 자동화
- ❌ 근로자 감시용 에이전트 (동의 없이)
- ❌ 급여/노무 관리 시스템 해킹

---

## 📋 고위험 사용 사례 (High-Risk Use Cases)

당신의 프로젝트가 아래에 해당하면 **추가 요구사항** 필요:

| 카테고리 | 당신의 상황 | 필수 조치 |
|---------|-----------|----------|
| **헬스케어** | 근로자 건강 데이터 수집 시 | ⚠️ 의료 전문가 검토 필수 |
| **고용/채용** | 출석 데이터로 성과 평가 | ⚠️ 인사팀 승인 필수 |
| **법률** | 계약서 자동 분석 제공 | ⚠️ 변호사 검토 필수 |
| **보험** | 근로자 배치 기반 보험료 산정 | ⚠️ 보험사 협의 필수 |

## 🤖 에이전트 사용 시 필수 공개 (Disclosure Rules)

### 팝아이즈를 외부에 공개할 경우:
```
필수 표시:
"이 출석 시스템은 AI 기술을 사용하여 보조합니다."
```

### Claude Forge 기반 자동화를 팀에 공개할 경우:
```
필수 고지:
"이 자동화는 Claude AI 에이전트를 사용합니다.
최종 승인은 인간이 수행합니다."
```

---

## 🔒 개인정보보호 체크리스트 (당신의 팝아이즈 앱)

### 데이터 수집
- [ ] 개인정보 수집 동의서 작성
- [ ] 목적명시: "출석 관리만" (다른 용도 금지)
- [ ] 수집 범위: 얼굴 이미지만 (이름, 급여 정보 연결 금지)

### 데이터 저장
- [ ] Turso DB 암호화 설정
- [ ] 접근 권한: PM/관리자만
- [ ] 백업 암호화

### 데이터 삭제
- [ ] 근로자 퇴직 후 30일 이내 삭제
- [ ] 또는 정기 자동 삭제 (90일 마다)

### 제3자 공개 금지
- [ ] 클라우드 API (AWS Rekognition 등) 전송 금지
- [ ] 오직 로컬 INFACE API 또는 Turso만 사용

## ⚠️ Claude 사용 금지 사항 (당신 상황에서)

### ❌ 절대 금지
```
1. Claude를 사용하여 근로자 감시 영상 자동 분석
   → 동의 없는 감시는 불법

2. 얼굴 인식으로 특정인 추적/위치 파악
   → 개인정보보호법 위반

3. 자동 커밋/배포를 사람 몰래 진행
   → "Human-in-the-loop" 위반

4. AI 생성 콘텐츠를 인간 작성으로 표기
   → 기만 행위

5. 경영진만을 위한 비밀 에이전트 운영
   → 투명성 부재
```

---

## ✅ 안전한 사용 패턴 (당신이 따르면 됨)

### 1️⃣ 개발 단계
```
Claude 코드 생성
  ↓
당신(인간)이 검토 & 수정
  ↓
Code-review 에이전트 검토
  ↓
Security-review 에이전트 검토
  ↓
최종 승인 후 배포
```

### 2️⃣ 배포 단계
```
자동화 에이전트 실행 (Forge)
  ↓
결과 로그 기록
  ↓
PM(당신)이 결과 검증
  ↓
이상 있으면 즉시 중단
```

### 3️⃣ 투명성 유지
```
팀 공지: "팝아이즈 v1.5부터 AI 출석 검증 추가됨"
팀 공지: "Claude Forge로 자동화 일부 구현"
```

---

## 📞 문제 발생 시

- **의심스러운 에이전트 동작**: 즉시 `/manual-confirm` 추가
- **개인정보 침해 우려**: Anthropic 사용자안전팀 연락 `usersafety@anthropic.com`
- **법률 자문 필요**: 변호사 상담 (고용/보험/의료 관련)
