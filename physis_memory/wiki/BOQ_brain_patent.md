# 📜 특허 및 핵심 기술 로직 (brain_patent.md)

## 💎 [v11.4] Patent C: 하이퍼-지배력 스나이퍼 (Hyper-Dominance Sniper)

### 1. 진화된 판정: Inverse Single-Ray (v11.3 -> v11.4)
- **이론적 근거**: 수조속에서 도장을 찍듯, 부재의 내부에서 외부로 레이를 쏘는 기존 방식은 복잡한 요철 구간에서 오차가 발생함.
- **v11.4 사양**: 0.1mm 허공(부재 외부)에서 부재 중심을 향해 **역방향으로 1회만 발사**(`Inverse Single-Ray`).
- **이점**: 연산 횟수를 50% 절감하면서도, 맞닿은 면의 조인트 여부를 100% 무결점으로 판정함.

### 2. 기하 타격: Face-Finder 스탬핑 (Stamping)
- **핵심 기술**: 레이캐스팅의 물리적 한계를 극복하기 위해, 간섭 예상 구간의 면(Face) 좌표를 직접 추출하여 '도장'을 찍듯 데이터를 각인함.
- **결과**: 소수점 단위의 미세한 틈새나 겹침도 `Dominance Hierarchy`에 따라 즉각적으로 공제(Subtract) 처리함.

### 3. 지배력 엔진 (Dominance Hierarchy)
- **서열 판정**: `Column(LV.1) > Wall(LV.2) > Slab(LV.3)`
- **로직**: 기둥(Column)과 벽체(Wall)가 만나는 지점은 기둥이 지배권을 가짐. 벽체의 해당 면은 자동으로 '하얀 속살(JOINT)' 재질로 치환되어 중복 물량 산출에서 제외됨.

### 4. 시각적 무결성: 인비저블 엣지 프리징
- **처리**: 조인트 판별 후 생성된 내부 경계선에 `soft/smooth/hidden` 속성을 주입하여 시각적으로는 매끄러운 단일 부재로 보이게 함 (물량 데이터는 보존).
