# 재무 분석 핵심 지표

[[주식_금융_경제_도메인]] | [[DART_시스템]] | [[Stock_AI_파이프라인]]

> Stock-AI의 DartClient가 가져오는 재무제표를 DeepSeek이 해석할 때 쓰는 지표들.

## 가치 평가 비율 (Valuation)

### PER (Price to Earnings Ratio)
```
PER = 주가 / EPS(주당순이익)
```
- "이 회사 1년 이익을 몇 년 모아야 시가총액인가"
- 한국 시장 평균 약 10~15배
- IT/성장주: 30~100배 (성장 기대 반영)
- 가치주: 5~10배

★ **단순히 낮다고 좋은 게 아니다** — 사양 산업은 PER 낮아도 미래 위험.

### PBR (Price to Book Ratio)
```
PBR = 주가 / BPS(주당순자산)
```
- "이 회사 청산 가치 대비 몇 배에 거래되나"
- 1.0 미만 = 청산가치 미만 (저평가? 또는 위기?)
- 한국 가치주에서 자주 사용 (금융주·건설주)

### PSR (Price to Sales Ratio)
```
PSR = 시가총액 / 매출
```
- 적자 회사도 평가 가능 (성장주에 유용)

### EV/EBITDA
```
EV = 시가총액 + 순부채
EBITDA = 이자/세금/감가상각 전 영업이익
```
- 부채까지 고려한 진짜 기업가치 평가
- M&A 평가에 자주 사용

## 수익성 비율 (Profitability)

### ROE (Return on Equity)
```
ROE = 순이익 / 자기자본
```
- 자본 효율성. 워런 버핏 1순위 지표.
- 15% 이상 우수
- 한국 평균 약 8~10%

### ROA (Return on Assets)
```
ROA = 순이익 / 총자산
```
- 자산 효율성. 자본 + 부채 합친 기준
- 5% 이상 우수

### 영업이익률
```
영업이익률 = 영업이익 / 매출
```
- 본업의 수익성
- 산업별 격차 큼 (반도체 30%+, 유통 5%, 건설 5~10%)

## 안정성 비율 (Stability)

### 부채비율
```
부채비율 = 부채총계 / 자기자본 × 100
```
- 200% 미만 안정 (한국 기준)
- 100% 미만 우수
- 금융업·항공업은 본질적으로 높음

### 유동비율
```
유동비율 = 유동자산 / 유동부채 × 100
```
- 단기 지급능력
- 200% 이상 우수, 100% 미만 위험

### 이자보상배율
```
이자보상배율 = 영업이익 / 이자비용
```
- 1.0 미만 = 영업이익으로 이자도 못 갚음 (좀비기업)
- 5.0 이상 안정

## 성장성 비율 (Growth)

### 매출성장률 / 이익성장률 (YoY)
```
성장률 = (당기 - 전기) / 전기 × 100
```
- 한국 시장: 10% 이상 성장주
- 4분기 연속 마이너스 = 추세 하락

## 현금흐름 (Cash Flow)

★ **이익보다 더 중요할 때 많음** — 회계 이익은 조작 가능, 현금은 거짓말 못함.

| 활동 | 의미 | 정상 |
|------|------|------|
| 영업 CF | 본업 현금 창출 | (+) |
| 투자 CF | 설비·M&A 등 | (-) |
| 재무 CF | 차입·배당 | 상황에 따라 |

**(+)(-)(-) 정상 패턴**: 본업 흑자, 투자 진행, 부채 상환·배당.
**(-)(+)(+) 위험 패턴**: 본업 적자, 자산 매각, 차입 의존.

## DART 자료 활용

[[DART_시스템]] 참조.
- 사업보고서 (3월 말 제출, 연간)
- 분기보고서 (1·2·3분기 종료 후 45일)
- 주요사항보고서 (수시 공시)
- 정정공시 ★ — 과거 수치 변경, 신뢰도 감점

## 피지수 학습 포인트

1. **단일 지표보다 비율 조합** — PER 낮은데 ROE 낮으면 그냥 안 좋은 회사
2. **업종 평균과 비교** — 절대값보다 동종업계 상대 비교
3. **추세 본다** — 분기/연간 변화율
4. **현금흐름 우선** — 회계 이익보다 현금이 정직
5. **공시 정정 이력** — 잦은 정정 = 신뢰도 감점

## Stock-AI DeepSeek 연동

DeepSeek은 재무 수치 추론 강점. DART에서 가져온 수치를 다음 형태로 분석:
- PER/PBR/ROE 등 핵심 비율 계산
- 동종업계 상대 위치
- 분기 추세
- 위험 신호 플래그 (이자보상배율 < 1, 부채비율 > 300% 등)
