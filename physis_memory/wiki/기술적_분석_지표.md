# 기술적 분석 핵심 지표

[[주식_금융_경제_도메인]] | [[Stock_AI_파이프라인]] | [[멀티AI_앙상블_원리]]

> Stock-AI의 ChartAgent (Gemini Vision)가 차트 이미지에서 읽어내는 것들.

## 캔들 (Candlestick)

**1봉 = 1단위 시간** (1분/5분/일/주/월)

```
    ┃    ← 고가 (High)
    █    ← 종가 (Close, 양봉이면 위)
    █
    █    ← 시가 (Open, 양봉이면 아래)
    ┃    ← 저가 (Low)
```

- **양봉 (빨강/파랑)**: 종가 > 시가 — 한국은 빨강
- **음봉 (파랑/빨강)**: 종가 < 시가 — 한국은 파랑
- **십자형 (Doji)**: 시가 ≈ 종가 — 추세 전환 가능

## 이동평균선 (MA / Moving Average)

| MA | 의미 |
|----|------|
| 5일선 | 단기 추세 (1주) |
| 20일선 | 중기 (1개월, 생명선) |
| 60일선 | 분기 추세 |
| 120일선 | 반기 추세 |
| 200일선 | 연간 추세 |

**골든크로스**: 단기선이 장기선을 상향 돌파 → 매수 신호
**데드크로스**: 단기선이 장기선을 하향 돌파 → 매도 신호

★ 단순한 매수/매도 신호가 아니라 *추세 확인* 도구로 봐야 함. 후행 지표.

## RSI (Relative Strength Index)

```
RSI = 100 - 100 / (1 + 평균상승폭/평균하락폭)
```
- 0 ~ 100 사이
- **70 이상 = 과매수** (매도 신호)
- **30 이하 = 과매도** (매수 신호)
- 다이버전스: 가격은 신고가인데 RSI는 못 따라가면 추세 약화

## MACD (Moving Average Convergence Divergence)

```
MACD line = 12일 EMA - 26일 EMA
Signal line = MACD의 9일 EMA
Histogram = MACD - Signal
```
- MACD가 Signal 상향 돌파 → 매수
- 0선 위에서 작동하면 상승추세
- 히스토그램 축소 = 추세 약화

## 볼린저 밴드 (Bollinger Bands)

```
중앙선 = 20일 MA
상단 = 중앙 + 2σ
하단 = 중앙 - 2σ
```
- 가격이 상단 터치 → 과매수 가능
- 하단 터치 → 과매도 가능
- 밴드 폭 좁아짐 (스퀴즈) → 큰 변동성 임박 신호

## 거래량 (Volume)

★ **가장 강력한 보조 지표**. 가격 신호의 진위 검증.
- 상승 + 거래량 증가 = 진짜 매수세
- 상승 + 거래량 감소 = 약한 상승 (의심)
- 하락 + 거래량 폭증 = 매도 클라이맥스 (반등 가능)

## 지지·저항 (Support / Resistance)

- **지지**: 가격이 떨어지다가 멈추는 가격대 (매수세 강함)
- **저항**: 가격이 오르다가 막히는 가격대 (매도세 강함)
- **돌파**: 거래량 동반해야 진짜 돌파
- **역할 전환**: 저항 돌파 후 → 그 가격대가 지지로

## Stock-AI ChartAgent (Gemini Vision) 학습

ChartAgent는 차트 이미지를 받아 Vision LLM이 읽음.
주요 출력 가설:
- 추세 방향 (상승/하락/횡보)
- 주요 지지·저항 가격대
- 보조지표 신호 (RSI/MACD/볼린저 위치)
- 거래량 패턴
- 캔들 패턴 (해머·도지·삼병·헤드앤숄더 등)

## 피지수 학습 포인트

1. **단일 지표 신호는 약함** — 여러 지표가 같은 방향이어야 신뢰도 ↑
2. **시간 프레임 정합** — 일봉 신호와 5분봉 신호 다를 수 있음
3. **거래량 = 진위 판단** — 거래량 없는 가격 변동은 페이크 가능
4. **후행 지표 한계** — 모든 기술적 지표는 과거 데이터 기반
