## 현재 상태 (2026-05-06)

- **`SITE.glb`** (635 KB) — 이천(3지국) 정사 모델. `complex_master.glb` 호혜.
  - 16개 동, 153 노드. 동별 노드 이름: `D{num}_B1` / `D{num}_B2`
  - 예: `D101_B2`, `D101_B1`, `D107_B1`, `D116_B1`
- **`01.pdf`** (144 KB) — 동위치/층수 라벨 평면도 (방부장 친람 자료)
- **`_archive/`** — 이전 SITE.gltf/SITE.bin (2026-05-05 작업분)

## 동 깜빡임 매칭 규칙 (`DrawingViewer.tsx`)

`highlightDong = "101동"` 입력 시, 다음 노드 이름 패턴이 모두 매칭됨:

- ✅ `D101_B2` / `D101_B1` (이천 GLB 형식)
- ✅ `101동`, `101동_벽체`, `Building_101동`
- ✅ `_101_…` (단어 경계가 D, _, 또는 시작인 경우)
- ❌ `L101_*` (라벨/대지 — 의도적 제외)
- ❌ `dong_101`, `b101` (소문자 prefix는 매칭 안 함)

---

## 모드 우선순위

1. **GLB**: `SITE.glb` / `SITE.gltf` / `site.glb` / `site.gltf` 자동 감지
2. **PDF**: `site.pdf` (현재 미사용)
3. **PNG**: `site.png` (현재 미사용)
4. **자동 격자**: 4×4 16동 25층 표준 박스. `BASEMENT_B2` 상수로 지하층수 구분.
