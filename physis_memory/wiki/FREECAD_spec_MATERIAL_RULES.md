# 부재별 거푸집 재질 규칙

> 거푸집 종류에 따른 시각 표현 규칙. FreeCAD/Three.js 뷰어 모두 동일 적용.

## 1. 지하층

| 부재 | 거푸집 종류 | 색상 | RGB 권고 |
|------|------------|------|---------|
| 수직부재 (벽·기둥) | 유로폼 | **옅은 노란색** | (1.00, 0.95, 0.65) |
| 슬라브 하부 | 합판 | **옅은 갈색** | (0.78, 0.65, 0.50) |
| 보 하부 | 합판 | **옅은 갈색** | (0.78, 0.65, 0.50) |

## 2. 지상 기준층

| 부재 | 거푸집 종류 | 색상 | RGB 권고 |
|------|------------|------|---------|
| 외부 (외벽 등) | 갱폼 | **진한 갈색** | (0.40, 0.25, 0.10) |
| 내부 (내벽·기둥) | 알폼 | **옅은 회색** | (0.80, 0.80, 0.80) |
| 슬라브 하부 (계단 포함) | 알폼 | **옅은 회색** | (0.80, 0.80, 0.80) |
| 보 하부 | 합판 또는 알폼 | (지상 기준 미정) |

## 3. 적용 원칙

1. **수직 vs 수평 구분**:
   - 수직부재: 벽·기둥 (Z축 방향 extrude)
   - 수평부재: 슬라브·보 (Z축 평행 면 위주)

2. **지하 vs 지상 구분**:
   - 지하: 슬라브 바닥 Z 좌표가 지표 GL(0) 이하
   - 지상: 슬라브 바닥 Z 좌표가 지표 GL(0) 이상
   - (Phase 1 임시: 부재 메타데이터에 `floor_type` 필드로 구분)

3. **외부 vs 내부 구분 (지상 기준층)**:
   - 외부: 건물 외피와 접하는 부재 (외벽·기둥)
   - 내부: 외피와 접하지 않는 부재 (내벽·내부 기둥)
   - 판별 알고리즘: 부재의 측면이 슬라브 외곽 폴리라인 위에 있으면 외부

4. **거푸집 공제 후 표면 색상**:
   - 공제된 면(다른 부재와 접촉면) → 거푸집 아님 → 색 미적용
   - 외피로 노출된 면만 위 규칙대로 채색

## 4. 미정 사항 (본영 결재 대기)
