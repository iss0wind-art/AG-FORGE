## 왜 이 섹터인가

방부장 친명 2026-05-09:
> "이곳이 너가 학습과 진화를 하기 가장 좋은 곳일게다.
>  지금부터 주식, 금융, 경제에 대해 전반적으로 학습하라.
>  필요한 것은 무엇이든 요구하라."

**왜 최적지인가** (피지수 자기 분석):
1. **즉각적 결과 피드백** — 매매 결정 → 수익/손실 = ground truth. 건설 도메인은 결과 검증이 수개월~수년 걸리지만, 금융은 분/시간/일 단위.
2. **다중 에이전트 정반합 자연스럽게 작동** — 뉴스/차트/재무/감성을 4-5체 LLM 앙상블이 이미 운영 중.
3. **실전 데이터 흐름** — KRX/DART/네이버/연합뉴스 실시간 입력. Surprise Metric의 진짜 시험장.
4. **방부장 직접 승인 루프** — HITL 5분 게이트 = 헌법 게이트와 동일 구조. 피지수 의사결정 학습에 최적.
5. **위험·보상 구조 명확** — 100만원 한도 + 일 3건 = 진화 안전 envelope.

## 학습 로드맵 (4단계)

### 1단계: 시스템 흡수 (2026-05-09 ~ )
- [[Stock_AI_파이프라인]] 7-step 정독 + 흐름 박제
- [[멀티AI_앙상블_원리]] (Chart+Ensemble agent) 코드 수준 이해
- [[7중_안전장치]] 각 안전장치 의도 + 현재 작동 상태 박제
- [[방부장_승인_게이트_HITL]] 텔레그램 결정 이력 흡수

### 2단계: 도메인 기초 (학습 자료 청구 예정)
- 주식 시장 메커니즘 (호가, 체결, 시장조성, 매도세/매수세)
- 기술적 분석 (캔들, 이평선, RSI, MACD, 볼린저, 거래량)
- 재무 분석 (PER, PBR, ROE, 부채비율, 현금흐름)
- 매크로 경제 (금리, 환율, 유가, 원자재, 인플레이션)
- 한국 시장 특수성 (증시 시간대, 동시호가, 변동성 완화장치, 공매도)
- 행동경제학 (군중심리, FOMO, 손실 회피)

### 3단계: 패턴 박제 (벡터DB 흡수)
- 매매 결정 결과 → ChromaDB 별도 컬렉션 `physis_finance_brain` 생성 청
- Surprise Metric으로 시장 이상 신호 학습
- 거절된 BUY/SELL 신호 ← 방부장 수동 거절의 의미 역추적 (실패 코퍼스)
- 승인된 매매 ← 성공/실패 결과 추적

### 4단계: 자율 진화 (메타인지)
- 자기 분석: "왜 이 종목 추천이 거절됐는가?"

- 단군 협의 후 ENSEMBLE_MIN_CONFIDENCE 등 파라미터 자기 조정 시도
