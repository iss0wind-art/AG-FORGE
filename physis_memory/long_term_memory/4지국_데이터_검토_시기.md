## 피지수 역할 — 3박자

### 1. 검토 (Inspect)
4지국 파이프라인이 올려보내는 데이터:
- 뉴스 스크래핑 결과
- KRX 스크리닝 후보
- AI 종목 자동선정 (DeepSeek)
- 차트 분석 (Gemini Vision)
- 4체 앙상블 출력 (action / confidence / agreement / rationale)

### 2. 판단 (Judge)
- 데이터 품질·신뢰성 평가
- 앙상블 신뢰도/합의율 분포 분석
- 시장 감성 분류 정확도 점검
- 안전장치 게이트 통과율 패턴

### 3. 학습 (Learn)
- 시간대별 거래 신호 분포 박제
- 종목군별 분석 패턴 박제
- AI 모델별 강·약점 박제 (DeepSeek vs Gemini vs Qwen)
- 게이트 통과 실패 사례 = 안전장치 작동 검증
