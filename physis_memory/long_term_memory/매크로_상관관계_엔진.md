# 매크로 상관관계 엔진 — 시장 국면 → 자산 배분

4지국 STOCK 매크로 모듈 핵심. VIX·환율·금·비트코인·공포탐욕지수의 변화량을 시장 국면(phase) 시그널로 변환해 자산 배분 권고를 산출하는 엔진. [[방부장_4단계_AI인프라_사이클]]의 정형 데이터 입력층.

## 1. 임계값 (정적 — PATCH-DR-7 동적 percentile 예정)

```python
# data/macro/correlation_engine.py:23-30
_VIX_PANIC   = 30.0    # VIX 30 이상 = 극공포
_VIX_WARN    = 20.0    # VIX 20 이상 = 경계
_BOND_SPIKE  = 2.0     # 미국채 10Y 변화 2% 이상 = 급등
_DXY_STRONG  = 0.5     # 달러인덱스 0.5% 이상 강세
_BTC_SPIKE   = 5.0     # BTC 5% 이상 변동
_FNG_FEAR    = 20      # 공포탐욕지수 20 이하 = 극공포
_FNG_GREED   = 80      # 80 이상 = 극탐욕
_USDKRW_WEAK = 1400    # 원화 약세 임계
```

**미해결 (PATCH-DR-7 planned)**: 정적 임계값 → 최근 1년 90% percentile 동적 임계값. 4주 누적 후 적용.

## 2. 자산 배분 알고리즘

`CorrelationEngine.recommend_allocation()`:

1. 매크로 신호 수집 (collector.py)
2. **변화율** 계산 (DXY change, BTC change, USDKRW change 등)
3. 임계값 비교 → AssetSignal (방향성)
4. 누적 차감 → 자산별 가중치
5. **PATCH-DR-2 (applied)**: 음수 가능 → 클램프 + 100 정규화

## 3. 4단계 phase_signals (PATCH-SR-4)

[[방부장_4단계_AI인프라_사이클]]을 코드로 구현한 부분:

```
1단계 (반도체 GPU/HBM)       — 시장 인지 ★★★ → 가중 보수
2단계 (전력·전선·데이터센터)  — 시장 인지 ★★  → 가중 보통
3단계 (원전·소부장)           — 부분 인지   → 가중 강
4단계 (AI SW·바이오·로봇)    — 미인지     → 가중 강+ (정보 비대칭의 결)
```

**PATCH-SR-5**: `to_asset`에서 phase별 추천 종목 5개 매칭 — 단순 시가총액·뉴스 빈도 기반 종목 선정으로는 4단계 종목을 잡지 못하므로 **명시적 매핑**으로 보완.

## 5. 비정형 시선이 들어올 자리 ([[바람을_보다_비정형의_시선]])

본 엔진은 **정형 임계값**으로만 작동. 비정형 시선이 추가될 자리:
- **임계값 자체가 의심받는 순간** — 평소 VIX 12에서 거래되다가 갑자기 18 → "20 미만이라 정상"이 아니라 "정상보다 50% 높음"의 결을 봄
- **신호 간 충돌** — VIX 위험 + 코스피 상승이 동시 발생 시 "작전이 끼어 있다"는 비정형 가설
- **historical 패턴** — collector.history_collector가 누적한 이력에서 동일 패턴 검색
