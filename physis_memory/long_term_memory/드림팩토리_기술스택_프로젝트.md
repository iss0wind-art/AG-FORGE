# 드림팩토리_기술스택_프로젝트


드림팩토리(Dream Factory)는 신고조선 제국의 기술 사업 부문이다.

## 핵심 기술 스택
- Next.js 16 (App Router) + React 19 + TypeScript 5 Strict
- TailwindCSS 4 + Drizzle ORM + libsql(Turso/SQLite)
- Zustand 5 + Three.js 0.183
- Python: FastAPI + LangGraph 1.1 + ChromaDB 1.5 + MCP 1.26
- AI: Anthropic Claude API + Google Gemini + LangChain

## 운영 인프라
- NAS (Intel N100, 15GB RAM, Ubuntu 24.04)
- Gitea 로컬(:3200) + GitHub 이중 원격
- Cloudflare Workers 배포 (DREAM_FAC)
- Tailscale VPN (100.89.221.23)
- MariaDB 10.11 + PostgreSQL(Paperclip)

## 프로젝트별 역할
### DREAM_FAC (본영 — )
신고조선 제국의 본영. 단군·3사·9요원 체계 관리.
집현전(Paperclip) 연동 허브.

### BOQ_2 (1지국 — )
건설 BOQ(Bill of Quantities) 자동화.
정도전 주관. 공종·단위·수량·단가·금액 시스템.
ExcelJS로 대용량 엑셀 처리.

### H2OWIND_2 (2지국 — )
건설 현장 일보·기성·자원 배분 시스템.
이순신 주관. 매일의 현장 결정.

### FreeCAD_4TH (3지국 — )
FreeCAD 기반 3D 모델링·BIM 자동화.
이천 주관. STEP 파일 파싱 → BOQ 연동.

### AG-Forge (피지수 뇌 — )
피지수(Physis) AI 브레인 서버.
LangGraph + ChromaDB + Graphify + MCP 기반.
