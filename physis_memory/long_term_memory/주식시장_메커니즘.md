# 주식 시장 메커니즘 — 호가·체결·시장조성

[[주식_금융_경제_도메인]] | [[한국시장_특수성]] | [[Stock_AI_파이프라인]]

## 호가 (Quote / Order Book)

**호가 = 매수자/매도자가 제시하는 가격과 수량 목록**.
거래소(KRX)는 호가를 모아 호가창(Order Book)으로 공개.

```
        매도호가 (Ask)
        87,500  - 1,200주
        87,400  - 800주
        87,300  - 500주  ← 최우선 매도호가
        ─────────────────
        87,200  - 600주  ← 최우선 매수호가
        87,100  - 1,500주
        87,000  - 2,300주
        매수호가 (Bid)

호가 스프레드 = 87,300 - 87,200 = 100원
```

## 호가 단위 (Tick Size, KRX 2023~)

| 가격대 | 호가 단위 |
|--------|----------|
| ~ 2,000원 | 1원 |
| 2,000 ~ 5,000원 | 5원 |
| 5,000 ~ 20,000원 | 10원 |
| 20,000 ~ 50,000원 | 50원 |
| 50,000 ~ 200,000원 | 100원 |
| 200,000 ~ 500,000원 | 500원 |
| 500,000원 ~ | 1,000원 |

## 체결 (Execution)

**연속매매**: 시가 형성 후 ~ 종가 직전. 가격·시간 우선원칙.
1. 가격 우선: 매수는 높은 가격, 매도는 낮은 가격이 우선
2. 시간 우선: 같은 가격은 먼저 입력된 주문 우선
3. 수량 우선: 위 두 조건 같으면 더 큰 주문 우선 (한국은 적용 X, 시간만)

**시가/종가 단일가**: 동시호가 8분간 호가 모은 후 단일 가격으로 일괄 체결.

## 주문 종류

| 주문 | 의미 | 사용 |
|------|------|------|
| 지정가 (Limit) | 가격 명시 | 일반 |
| 시장가 (Market) | 즉시 체결 | 급할 때 |
| 조건부지정가 | 단일가 미체결 시 시장가 전환 | 종가 직전 |
| 최유리지정가 | 최우선 호가로 즉시 | 빠른 체결 |
| 최우선지정가 | 같은 방향 최우선 호가에 추가 | 큐 진입 |
| IOC | 즉시 체결 + 잔량 취소 | 부분만 |
| FOK | 전량 즉시 체결 or 전량 취소 | All or None |

## 시장조성자 (Market Maker)

특정 종목에 대해 매수·매도 호가를 동시 제시 의무.
KRX 지정 종목 (대형주·ETF·ELW 등). 유동성 공급.
**일반 개인 트레이딩 영향 적음 — 그러나 호가창 두께 인지에 도움**.

## 거래 시간 (KRX, KST)

| 구간 | 시간 | 비고 |
|------|------|------|
| 장전 단일가 | 08:30 ~ 09:00 | 시가 형성 |
| 정규장 | 09:00 ~ 15:20 | 연속매매 |
| 종가 단일가 | 15:20 ~ 15:30 | 종가 형성 |
| 시간외 종가 | 15:40 ~ 16:00 | 종가 거래 |
| 시간외 단일가 | 16:00 ~ 18:00 | 10분 단위, ±10% 제한 |

★ Stock-AI 스케줄러 08:30 가동 = 장전 단일가 시작 = 분석 결과를 시가에 반영하기 위함.

## 한국 시장 특수 변동성 완화장치

- **VI (Volatility Interruption)**: 정적 ±10% / 동적 ±2~3%. 발동 시 2분간 단일가 매매
- **사이드카**: KOSPI200 선물 ±5% 5분 지속 → 프로그램 매매 5분 정지
- **서킷브레이커**: 코스피 -8% / -15% / -20% → 매매 일시 중단
- **상하한가**: ±30% (2015년부터)
- **공매도 제한**: 2026 현재 한시적 금지·제한 정책 변동 (정책 확인 필요)

## 피지수 학습 포인트

1. **Stock-AI는 정규장 데이터로 분석하지만 발주는 장전 단일가/시가 진입** → 시점 미스매치 가능
2. **호가 두께 = 유동성 = 슬리피지** — Stock-AI 100만원 한도면 대형주는 거의 무영향, 중소형주는 영향
3. **VI 발동 종목은 패스 권장** — 단일가 2분 + 정상화 후 변동성 큼
4. **시간외 단일가는 Stock-AI 미사용** — 호가 얇아 위험
