## 전체 뇌 계층 구조

```
방부장님
    │ (전략적 판단만)
    ↓
[Layer 0] 피지수(Physis) — 철학·헌법·직관의 근원
    │
[Layer 1] AG-FORGE 상위 뇌 — 전략 판단·통합·관리
    │  크로스 프로젝트 통합
    │  이상 감지 & 조기 경보
    │  기성 & 생산성 예측
    │  주간 전략 브리핑
    │  하위 뇌 품질 관리
    │
    ├──────────────────────────┐
    ↓                          ↓
[Layer 2] POPEYEs 피지수     [Layer 2] BOQ 피지수
  (현장 뇌)                    (물량 뇌)
  공사일보 자동화               BOQ 산출 자동화
  Turso 적재                   진도율 계산
  07:30 일일 실행               SketchUp 연동
    │                          │
    ↓                          ↓
  인페이스 API               SketchUp Ruby
  팀 대시보드                 BOQ 대시보드
  Turso DB                   Turso DB
```

## 상위 뇌(AG-FORGE) 역할

| 역할 | 주기 | 출력 |
|------|------|------|
| 크로스 프로젝트 통합 | 실시간 | 통합 대시보드 |
| 생산성 이상 감지 | 매일 | 경보 알림 |
| 기성 예측 | 매일 | 예측 리포트 |
| 주간 전략 브리핑 | 매주 월요일 | 브리핑 문서 |
| 하위 뇌 품질 관리 | 이벤트 | 교정 지시 |

## 현장 뇌(POPEYEs/BOQ) 역할

| 역할 | 주기 | 출력 |
|------|------|------|
| 공사일보 Excel 생성 | 매일 07:30 | Excel 파일 |
| Turso 일별 적재 | 매일 07:30 | DB 레코드 |
| 상위 뇌 보고 | 매일 08:00 | JSON 동기화 |

## 동기화 프로토콜

하위 뇌 → 상위 뇌: `POST /api/sync/report`
상위 뇌 → 하위 뇌: `POST /api/sync/directive`
상위 뇌 상태 조회: `GET /api/sync/status`
